# Wedding Invitation Website

Premium editable Indian wedding invitation website.

## Features
- Ganesh welcome opening screen
- Curtain opening animation
- Lord Venkateswara Swamy hero background
- Editable names, date, venue, timings, phone numbers, story and family details
- Photo upload gallery
- Countdown timer
- Google Maps venue section
- WhatsApp RSVP button
- Mobile responsive design

## Publish
Use GitHub Pages from the `main` branch and `/root` folder.
